---

## 📦 Assignments & Implementations

### 1. 🕷️ Web Crawler (`Crawler.py`)
A simple web crawler that fetches a webpage and extracts:
- **Page Title**
- **Body Text**
- **All Hyperlinks**

**Usage:**
```bash
python scraper.py <url>
```

**Example:**
```bash
python scraper.py https://example.com
```

## ⚙️ Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Janu-Chaudhary/SEIR_Projects
cd SEIR_Projects
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

**`requirements.txt`**
```
requests
beautifulsoup4
```
